2. 棋子摆放

规则说明：在5×5棋盘上摆放棋子，考虑棋子的旋转和覆盖能力。
在5×5棋盘放置L型棋子（占3格），棋子可旋转，求最多放置多少个？
点击显示答案
7个（棋盘共25格，每个L型占3格，7个占21格）
一个棋子旋转几种不同姿态？
点击显示答案
4种旋转姿态（0°、90°、180°、270°）
在5×5棋盘上，每枚棋子能控制所在的行、列和两条对角线，问：最少需要放几枚这样的棋子，能让所有格子都被控制？
点击显示答案
5枚（通过放在对角线位置可覆盖全棋盘）
用占4格的Z型棋子（可镜像+旋转）覆盖5×5棋盘，求最少剩余格子数
点击显示答案
1格（25 ÷ 4 = 6个Z型棋子覆盖24格，剩余1格）

2. 成绩比较

解题方法：根据题目描述建立数学关系，解方程求未知数。
考试占比：期中30%，期末70%。甲期中80分期末90分，乙期中85分期末88分，谁最终成绩高？
点击显示答案
甲成绩: (80×0.3) + (90×0.7) = 24 + 63 = 87分
乙成绩: (85×0.3) + (88×0.7) = 25.5 + 61.6 = 87.1分
乙成绩略高
阿明得分是阿强3倍，阿强比阿明少15分，求阿明得分。
点击显示答案
设阿强得分x，则阿明得分3x
3x - x = 15 → 2x = 15 → x = 7.5
阿明得分: 3×7.5 = 22.5分
甲从70分升到80分，乙从85分升到90分，谁进步更大？
点击显示答案
甲提升: 80 - 70 = 10分
乙提升: 90 - 85 = 5分
甲进步更大
